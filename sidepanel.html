Welcome to SarvaMitra, your companion!

Navigation -

  1. Use Tab key to navigate around chunks.
  2. Use space or enter keys to select and unselect chunks.

Use the shortcuts below to use tools.

  1. Ctrl+Shift+1 ‚Üí Turn microphone on/off
  2. Ctrl+Shift+2 ‚Üí Send query
  3. Ctrl+Shift+3 ‚Üí Read Aloud
  4. Ctrl+Shift+4 ‚Üí Translate

Tips:

  1. Copy - Click a message or focus textbox, then use this to copy content
  2. Images: Click on images below to select them for context.
  3. Selected images will have blue borders.
